# March Madness Upset Analysis

## Overview
This project analyzes historical March Madness data to identify patterns related to upset potential and tournament success. The analysis focuses on three main ideas: when upsets occur most often, how seeding affects Final Four probability, and whether offensive efficiency is associated with deeper tournament runs.

## Key Questions
- Which rounds produce the most upsets?
- How strongly does seed predict Final Four success?
- Does offensive efficiency help explain tournament success?

## Tools Used
- Python
- Pandas
- Jupyter Notebook / VS Code
- Tableau
- CSV datasets

## Main Findings
- Most upsets occur in the early rounds, especially the First Round.
- Lower-numbered seeds have a much higher probability of reaching the Final Four.
- Teams with higher offensive efficiency are more likely to make deeper tournament runs.

## Repository Structure
- `data/` contains raw and processed datasets
- `notebooks/` contains the analysis notebook
